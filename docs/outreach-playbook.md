# Non-Social Outreach Playbook

Use this file for weekly email outreach. The goal is links, mentions, and qualified tool inclusion without social posting.

## Weekly volume

- Send `10-15` high-quality emails per week
- Follow up once after `5-7` days
- Drop the target after the second unanswered touch

## Best target types

- bookkeeping and accounting resource pages
- freelancer tools roundups
- software directories that accept editorial resources or tools
- accountant / bookkeeper tool pages
- finance or operations pages where `statement-to-csv` is a real fit

## Three outreach angles only

### 1. Resource page inclusion

Use this when the target page already links to helpful accounting, bookkeeping, or freelancer tools.

Pitch:
- `statement-to-csv` first
- then one truly relevant roundup or review only if it fits the page

### 2. Broken-link replacement

Use this when a target page has a dead link to a banking, bookkeeping, or statement-conversion resource.

Pitch:
- the closest matching live resource
- usually `statement-to-csv`
- never force a review page where a utility or guide belongs

### 3. Tool value-add inclusion

Use this when the target page curates useful tools for accountants, bookkeepers, operations teams, or freelancers.

Pitch:
- `statement-to-csv` as a local-first utility
- mention the no-upload angle and bookkeeping cleanup use case

## Qualification rules

Only pitch pages that satisfy at least one of these:

- relevant audience overlap
- real existing resource list
- obvious utility-tool fit
- credible site and public contact route

Skip:

- spammy paid directories
- generic guest-post farms
- pages with no real audience fit
- sites that only exist to sell placements

## Default asset to pitch

Pitch `https://accountanttoolkit.com/statement-to-csv` first.

Why:

- easier to justify than a commercial review
- linkable utility angle
- useful to accountants, bookkeepers, and finance operators
- consistent with a no-social growth strategy

## Supporting assets to mention second

Only after the tool, use:

- `https://accountanttoolkit.com/best-bookkeeping-software-for-solo-accountants`
- `https://accountanttoolkit.com/best-receipt-tracking`
- `https://accountanttoolkit.com/dext-vs-hubdoc`
